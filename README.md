# Statistics and Data Management Code repository

This repository holds all code used in the KU Leuven courses [Statistiek en databeheer](https://onderwijsaanbod.kuleuven.be/syllabi/n/T2ASD1N.htm#activetab=doelstellingen_idp1516816) and [Statistics and Data Management](https://onderwijsaanbod.kuleuven.be/syllabi/e/T2ASD2E.htm#activetab=doelstellingen_idp634480).

# Usage

The files should be accessed via the JupyterHub link provided on the Toledo page of the course.
